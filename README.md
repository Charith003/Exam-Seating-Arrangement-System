*Introduction:* 
A desktop Java application called the Exam Seating Arrangement System was created to automate and simplify exam hall seating management. Constructed with MySQL for database connectivity and Java Swing for the graphical user interface, the system offers a quick and easy way to arrange students and exam rooms.

Admin and Student are the two primary roles available in the application. Administrators can create structured seating arrangements, add or remove rooms, manage student records, and specify seating capacities by logging in. Students can view their assigned seat and room details instantly by logging in with their roll numbers.

This project demonstrates how database administration, GUI design, and fundamental Java programming ideas can all be combined to create a useful academic solution.
